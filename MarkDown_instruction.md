## Списки

Чтобы создать ненумерованные списки, необходимо пункты выделить звездочкой (*)

Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы создать нумерованные списки, необходимо пункты просто пронумеровать

Например, вот так:
1. Первый пункт
2. Второй пункт
3. Третий пункт
