# Инструкция по работе с MarkDown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочкой (*)

Например, *вот так*.

Чтобы выделить текст полужирным необходимо обрамить его двуми звездочками (**)

Например, **вот так**.
